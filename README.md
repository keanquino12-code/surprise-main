# Surprise Main

A small HTML project with interactive surprise pages: `index.html`, `cake.html`, `letter.html`, and `memorylane.html`.

## How to use

1. Open `index.html` in Google Chrome.
2. If needed, right-click the file and choose "Open With" → "Google Chrome", or drag the file into a Chrome window.
3. Use the links on the page to explore the other surprise pages.

## Files

- `index.html` - main landing page
- `cake.html` - surprise cake page
- `letter.html` - surprise letter page
- `memorylane.html` - memory lane page
